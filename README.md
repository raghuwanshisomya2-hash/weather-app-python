# Weather App (Python)

This is a simple Python-based Weather Application that fetches real-time weather data using the OpenWeatherMap API.

## Features
- Search weather by city name
- Displays temperature in Celsius
- Shows humidity and weather condition
- Uses real-time API data

## Technologies Used
- Python
- Requests library
- OpenWeatherMap API

## How to Run
1. Clone the repository
2. Install requests library:
   pip install requests
3. Get an API key from https://openweathermap.org
4. Add your API key in `weather_app.py`
5. Run the program:
   python weather_app.py
